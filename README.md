# Currículo em HTML e CSS

Este projeto consiste na criação de um currículo profissional utilizando HTML e CSS. O objetivo é apresentar informações pessoais, acadêmicas e profissionais de forma organizada, moderna e visualmente atrativa.

O layout foi desenvolvido em duas colunas, com foco em um design limpo e profissional, destacando seções como experiência, habilidades, formação e contatos. Também foram utilizados ícones e tipografia personalizada para melhorar a aparência e a usabilidade.

Por ser um projeto estático, o código é simples, leve e fácil de personalizar, podendo ser utilizado como currículo digital ou base para um portfólio pessoal.
